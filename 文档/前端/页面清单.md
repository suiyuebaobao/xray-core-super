## 管理后台

| 路径 | 页面 | 说明 |
|------|------|------|
| `/admin/login` | `pages/admin/Login.vue` | 管理员登录 |
| `/admin` | `pages/admin/Dashboard.vue` | 仪表盘 |

| `/admin/relays` | `pages/admin/Relays.vue` | 中转节点、监听端口与出口节点绑定、一键部署中转，支持多选批量删除 |
| `/admin/users` | `pages/admin/Users.vue` | 用户管理、列表展示套餐与剩余流量、订阅管理、用量记录，支持多选批量删除非当前登录用户 |
| `/admin/orders` | `pages/admin/Orders.vue` | 订单管理 |
| `/admin/redeem-codes` | `pages/admin/RedeemCodes.vue` | 兑换码管理，生成时按套餐名称选择，列表展示套餐名称 |
| `/admin/subscription-tokens` | `pages/admin/SubscriptionTokens.vue` | 订阅 Token 管理 |

## 页面与接口对齐

所有页面调用后端接口时必须经过 `frontend/src/api/index.js` 适配层。当前对齐关系如下：

| 页面 | 主要接口 |
|------|----------|
| 用户登录/注册 | `POST /api/auth/login`、`POST /api/auth/register`、`POST /api/auth/refresh`、`POST /api/auth/logout` |
| 用户首页/订阅详情 | `GET /api/user/me`、`GET /api/user/subscription`、`GET /api/user/usage`、`GET /sub/:token/:format` |
| 用户套餐页 | `GET /api/plans`；支付购买流程暂不接入 |
| 用户兑换码 | `POST /api/redeem` |
| 用户资料 | `GET /api/user/me`、`PUT /api/user/profile`、`PUT /api/user/password` |
| 管理仪表盘 | `GET /api/admin/dashboard/stats` |
| 套餐管理 | `GET/POST/PUT/DELETE /api/admin/plans`、`GET/POST /api/admin/plans/:id/node-groups`；列表使用 `is_default` 与 `node_groups` 摘要展示基础套餐和绑定分组，基础套餐禁用删除，多选批量删除复用单条删除接口 |
| 节点分组 | `GET/POST/PUT/DELETE /api/admin/node-groups`、`GET/PUT /api/admin/node-groups/:id/nodes`；列表使用 `nodes.line_mode` 与中转后端绑定展示组内出口节点、直连/中转类型和中转线路数，多选批量删除复用单条删除接口 |
| 节点管理 | `GET/POST/PUT/DELETE /api/admin/nodes`、`POST /api/admin/nodes/deploy`、`POST /api/admin/nodes/deploy/scan-ips`；管理出口节点，支持 `transport=tcp/xhttp`、XHTTP path/mode/host、`line_mode` 控制直连/中转下发，多选批量删除复用单条删除接口 |
| 中转节点管理 | `GET/POST/PUT/DELETE /api/admin/relays`、`POST /api/admin/relays/deploy`、`GET/PUT /api/admin/relays/:id/backends`；列表展示中转入口、监听端口、绑定出口节点和健康状态，多选批量删除复用单条删除接口 |
| 用户管理 | `GET/POST/DELETE /api/admin/users`、`PUT /api/admin/users/:id/status`、`PUT /api/admin/users/:id/password`、`GET/PUT /api/admin/users/:id/subscription`、`GET /api/admin/users/:id/usage`；列表直接展示 `plan_name`、订阅状态、已用流量和剩余流量，支持表格多选批量删除 |
| 订单管理 | `GET /api/user/orders`、`GET /api/admin/orders`；支付回调暂不接入 |
| 兑换码管理 | `GET/POST /api/admin/redeem-codes`、`GET /api/admin/plans`；生成和列表按套餐名称展示，提交时仍使用套餐 ID |
| 订阅 Token 管理 | `GET/POST /api/admin/subscription-tokens`、`POST /api/admin/subscription-tokens/:id/reset`、`POST /api/admin/subscription-tokens/:id/revoke`；列表按用户显示唯一 Token、订阅链接、Token 状态、最后使用、Token 过期时间、套餐名称、套餐到期和流量用量，并提供 Clash/Base64/URI 完整订阅链接复制 |
