## 中转联调结果

以下记录为 2026-04-29 的历史中转验证。2026-05-05 后，154.219.106.105 已切换为多出口出口节点；生产库中 relay/backend 已停用，订阅不再下发该中转线路。

### 1. 一键部署 relay

- 部署入口：`POST /api/admin/relays/deploy`
- 中转入口：节点 2
- relay agent：`raypilot-relay-agent` 容器，`AGENT_ROLE=relay`
- 转发组件：容器内 HAProxy
- 回连状态：`relays.status=online`，`last_heartbeat_at` 持续更新

### 2. 后端绑定

| 字段 | 值 |
|------|----|
| relay | 节点 2 |
| listen_port | 24443 |
| exit_node | 节点 1 |
| target | `154.219.97.219:443` |
| 任务状态 | `relay_config_tasks.status=DONE` |

HAProxy 生成的核心配置：

```haproxy

  bind *:24443
  mode tcp
  default_backend backend_1

backend backend_1
  mode tcp
  server exit_1 154.219.97.219:443 check
```
