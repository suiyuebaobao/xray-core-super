# 2026-05-04 日报

## 今日变更

- 一键部署出口节点新增“多 IP 服务器”手动开关；默认单 IP 模式不扫描出口 IP。
- 新增多 IP 扫描接口，只有管理员开启多 IP 模式后才通过 SSH 读取公网 IPv4，并用 `curl --interface <IP>` 验证真实出口。
- 新增 `node_hosts` 物理节点服务器模型；多 IP 模式下一台服务器只运行一个 `AGENT_ROLE=multi_exit` node-agent，每个公网出口 IP 创建一条逻辑 `nodes`。
- node-agent 新增 multi_exit 模式，生成多 VLESS Reality inbound 与多 freedom outbound，并为每个逻辑节点绑定 `listen` 和 `sendThrough`。
- multi_exit 按 `node_id` 分别领取任务、上报任务结果和流量；Xray 本地使用节点前缀 email 分隔统计，上报中心前还原为原始 `xray_user_key`。
- 管理后台一键部署弹窗新增多 IP 开关、出口 IP 扫描和手动勾选确认。
- 一键部署新增旧节点代理清理步骤，部署前清理旧 systemd node-agent、旧二进制、旧容器并停止宿主机 Xray 端口占用。
