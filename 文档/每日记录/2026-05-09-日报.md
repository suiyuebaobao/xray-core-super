- 多 IP / multi_exit 端口冲突判定从“端口全局唯一”修正为“监听端点 `listen_ip:port` 唯一”，允许不同公网出口 IP 复用 443，同时继续阻止同一 IP 端口冲突；管理 API 新增和编辑节点也补充后端端点冲突校验，覆盖同一 `node_host_id`、同一 `agent_base_url` 和通配监听。

- 销售首页后台配置链接增加后端归一化和前端渲染双重白名单，只允许站内路径、锚点、HTTP 与 HTTPS，避免 `javascript:` 等危险链接进入公网首页。
