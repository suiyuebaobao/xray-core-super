# 接口文档 - 套餐接口

## 获取上架套餐列表

**GET /api/plans**

获取所有上架的套餐，供用户浏览和选择。

### 鉴权

公开接口，无需登录。

### 查询参数

无。

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 套餐 ID |
| name | string | 套餐名称 |
| price | number | 价格 |
| currency | string | 货币类型，默认 USDT |
| traffic_limit | number | 普通流量限制（字节） |
| residential_traffic_limit | number | 家宽流量限制（字节） |
| duration_days | number | 时长（天） |
| sort_weight | number | 排序权重，越小越靠前 |
| is_active | boolean | 是否上架 |
| is_default | boolean | 是否基础套餐 |
| created_at | string | 创建时间 |
| updated_at | string | 更新时间 |
